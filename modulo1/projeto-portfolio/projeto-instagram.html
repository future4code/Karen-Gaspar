Projeto Nova Interface do Instagram

>

Esse projeto foi desenvolvido em um bootcamp realizado junto a Digital Innovation One. A proposta foi recriar a interface de login do Instagram. Criei uma página de login com fundo preto, relembrando o modo "escuro" que muito usuários utilizam no aplicativo e adicionei uma imagem que mostra a funcionalidade "Reels" que é o que está em alta na rede.

Tecnologias Utilizadas:

  • HTML
  • CSS
