Projeto Loja de Utensílios de Chá

Esse projeto foi desenvolvido em aula junto a turma Carver na Labenu, onde nos foi proposto criar o site para uma loja de produtos de escolha livre. A proposta, além de criar a página inicial de apresentação da loja, consiste em vincular outras páginas. Uma contendo os produtos, valores e estoque, outra com formulário de compra e outra com nome dos funcionários. Foi um projeto bem divertido de elaborar!

Tecnologias Utilizadas:

  • HTML
  • CSS
