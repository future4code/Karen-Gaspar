Projeto Criação de Jogo da Cobrinha

Esse projeto foi desenvolvido em um bootcamp realizado junto a Digital Innovation One. A proposta foi criar o famoso Jogo da Cobrinha, onde essa vai aumentando conforme encosta no quadradinho avulso na tela que muda de lugar a cada vez que esse processo ocorre. O jogo termina quando a cobrinha encosta em alguma parte dela mesma, trazendo a mensagem de GAME OVER na tela.

Tecnologias Utilizadas:

  • HTML
  • CSS
  • JavaScript
